## 42 Tokyoとは

42 Tokyoは、フランス発のプログラミングスクールです。教師がおらず、学費も無料という革新的な教育モデルが特徴です。

## ピアラーニングの仕組み

42 Tokyoでは、学生同士が互いに教え合う「ピアラーニング」を採用しています。

### コードレビュー（ピア評価）

プロジェクトを提出すると、ランダムに選ばれた別の学生がコードをレビューします。

```c
#include <stdio.h>

int main(void)
{
    printf("Hello, 42 Tokyo!\n");
    return (0);
}
```

### メリット

1. **能動的な学習** — 人に説明することで理解が深まる
2. **多様な視点** — 同じ問題に対して複数のアプローチを学べる
3. **コミュニケーション力** — 技術的な議論を通じてスキルアップ

## まとめ

ピアラーニングは最初は戸惑いますが、慣れると非常に効果的な学習方法だと感じています。
